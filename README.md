# Barbershop_V
💈 Barbershop Website

Современный веб-сайт барбершопа, разработанный в качестве учебного проекта.
Проект демонстрирует навыки фронтенд-разработки, создание адаптивного интерфейса и работу с веб-технологиями.

🚀 О проекте

Этот проект представляет собой веб-сайт барбершопа, где пользователи могут:

просматривать услуги
узнавать информацию о барбершопе
смотреть галерею работ
находить контактную информацию

Цель проекта — практика разработки пользовательских интерфейсов и структуры веб-сайта.

Такие сайты обычно включают разделы с услугами, галереей и возможностью записи, что помогает барбершопам привлекать клиентов через интернет.

🛠 Используемые технологии

Проект разработан с использованием следующих технологий:

HTML5
CSS3
JavaScript
Responsive Design
